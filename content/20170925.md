如何建立自己的倉儲 以及用Leo管理自己的網誌

複習week2指令

mkdir 資料夾名稱 : 建立資料夾 
cd 資料夾名稱 or .. : 移動到指定資料夾 
clone 網址 : 克隆網址到自己的資料夾 
open 檔名 指定位置 : 打開資料夾內的資料至指定位置 add . : 新增全部檔案
 commit -m "名稱" : 提交
